# Module-for-Fuzzy-Logic-Using-Python
TYPE 2 Fuzzy Logic Tool created for Researchers 
Contains following functions 

**trilmf** : Triangular Lower Member function for membership value less than 1(h)
**triumf** : Triangular Lower Member function for membership value less than 1(h)
**umf** : Upper Triangular Membership Function 
**lmf** : Lower Membership Function 
**plotting_mf** : Plotting of membership functions
**interp** : Fuzzification of input
**user_rules** : Build rules
**output_plot** : Plots the output membership activities
**defuzzification** : Converts the fuzzy values to crisp values through determination of centroid
**centroid_plot** : Plots the aggregated result

Python Module Used:
**Numpy**
**skfuzzy**
**Matplotlib**
